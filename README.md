# TravelVN - Website Du Lịch Việt Nam

Website quảng bá và giới thiệu các tour du lịch Việt Nam, xây dựng bằng HTML, CSS và JavaScript thuần. Hệ thống hoạt động theo mô hình SPA (Single Page Application).

----------------------------------

## Cách chạy dự án

1. Tải dự án về máy:
- Clone bằng Git:
git clone https://github.com/bincasau/IE104_FE.git

2. Mở dự án bằng VS Code hoặc IDE bất kỳ.

3. Chạy file index.html:
- Tại file gốc
- Chuột phải -> Open with Live Server
hoặc
- Mở trực tiếp file index.html bằng trình duyệt.
Không cần backend, không cần cài đặt thêm.

----------------------------------

## Cấu trúc dự án

IE104_FE/
- pages/ (các trang Home, Tour, Blog, About, Contact)
- components/ (header, footer)
- assets/
    - css/
    - js/
    - images/
    - pdf/
- data/ (tours.json, blogs.json)
- lang/ (en.json, vi.json, cn.json, jp.json)
- index.html (file chính của SPA)

----------------------------------

## Tính năng chính

- Hỗ trợ 4 ngôn ngữ: Việt, Anh, Trung, Nhật
- Tìm kiếm tour và blog
- Lọc và sắp xếp tour
- Xem chi tiết tour và blog
- Lazy load hình ảnh
- SPA chuyển trang không tải lại
- Hoạt động tốt trên PC, tablet và mobile
